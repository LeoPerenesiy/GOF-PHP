- 👉 Proxy = обёртка над объектом, которая контролирует доступ к нему
- Основные кейсы:

1) 🐢 Lazy loading (ленивая загрузка)

Не создаём тяжёлый объект сразу
👉 создаём только когда реально нужен

2) 🔐 Контроль доступа

Проверка:

прав
токенов
ролей
3) 🌐 Удалённый доступ

Ты думаешь что работаешь с объектом
👉 а он на другом сервере
