--  Abstract Factory
-------------------------
- решает проблему создания связанных объектов,
- гарантируя их совместимость и убирая зависимость от конкретных классов
- 👉 Она нужна, когда у тебя есть разные “режимы системы”,
- и в каждом режиме нужно создавать связанный набор классов, которые нельзя смешивать.
