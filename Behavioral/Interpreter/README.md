- Паттерн интерпретатора GoF выполняет выражения непосредственно из 
- объектно-ориентированного абстрактного синтаксического дерева (AST),
- в то время как движок Zend Engine в PHP сначала компилирует AST
- в опкоды, а затем выполняет их на виртуальной машине.

- 👉 это когда ты делаешь свой мини-язык внутри кода и говоришь:
- “я буду писать правила, а программа их поймёт и выполнит”
