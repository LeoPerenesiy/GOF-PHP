- Memento = сделать “сохранение состояния” объекта и потом уметь откатиться назад
- Originator — основной объект (например редактор)
-  Memento — снимок состояния (как save-файл)
-  Caretaker — тот, кто хранит (например стек undo)
- это паттерн, который позволяет сохранить и восстановить состояние объекта без
- нарушения инкапсуляции. Часто используется для undo/redo.
- Примеры: тразакция
