🧠 Аналогия из жизни

💬 Mediator = продюсер / режиссёр

Актёры (объекты):

не общаются друг с другом
общаются только с режиссёром

Режиссёр решает:

кто говорит
кто молчит
что происходит дальше

- Events + Listeners (самое близкое)
- В Laravel нет отдельного паттерна Mediator, но его роль выполняет Event Dispatcher, который позволяет объектам 
- взаимодействовать косвенно через события, снижая связанность системы.
